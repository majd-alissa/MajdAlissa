About the Game


It is the first game of its kind in the Middle East, which is characterized by the presence of ancient Arab civilizations that you have to discover and at the same time linked to the Metaverse, In addition, this is the first game that supports the (NFT) system, which you can get it by passing the stage or doing the activities in the game or When you reach a certain place.

NFT means non-fungible tokens (NFTs), which are generally created using the same type of programming used for cryptocurrencies. In simple terms these cryptographic assets are based on blockchain technology.
